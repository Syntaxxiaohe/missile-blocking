## 今日完成工作清单

### P0 - M1 先验撕裂逻辑极致化
**文件**: `M1_NaiveGA/optimizer/distributed_optimizer.py`

**问题**: M1 原本的烟雾弹散布策略是"宽时间窗内均匀分布"，这给了 M1 广撒网碰运气的机会，无法体现"先验撕裂"导致的防线崩溃。

**修复内容**:
```python
# 修复前（Line 205-211）：宽时间窗内均匀分布 - 给M1作弊机会！
t_min, t_max = self.expert_gene["time_window"]
time_span = t_max - t_min
for k in range(SMOKE_PAYLOAD_PER_UAV):
    t_base = t_min + time_span * k / max(SMOKE_PAYLOAD_PER_UAV - 1, 1)
    individual[4 + k] = min(t_base, t_max)

# 修复后：在错误中点附近密集投放 - 彻底剥夺碰运气可能！
t_min, t_max = self.expert_gene["time_window"]
t_mid = (t_min + t_max) / 2.0
offsets = [-2.0, 0.0, 2.0]  # 密集投放，全部集中在错误的中点附近
for k in range(SMOKE_PAYLOAD_PER_UAV):
    t_val = t_mid + offsets[k % len(offsets)]
    individual[4 + k] = np.clip(t_val, t_min, t_max)
```

**理论意义**: 这一修改彻底剥夺了 M1 广撒网碰运气的可能，确保其因"先验撕裂"（错误的时间窗 + 错误的导弹匹配）而防线崩溃的理论预期得以完美实现。

| 特性 | M1_NaiveGA | M2_Clustering | M3_AmmoPenalty |
|------|-----------|---------------|----------------|
